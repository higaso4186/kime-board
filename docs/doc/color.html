キメボード カラーパレット（Sea 基準）

ベース色は lightseagreen (#20B2AA)。色数を絞り、Sea系トーン + Neutral + 最小のDangerで構成します。

1. 単純パレット

Sea トーン（主要UIの基準色）

sea-50#EAF9FA
sea-200#B8EBE8
sea-400#5ECFC8
sea-600#20B2AA
sea-800#167975

2. 文字色パターン

濃色の上は白文字を標準とする

Primary Button

背景: sea-600 / 文字: white

Secondary Button

背景: sea-50 / 文字: sea-800

Danger Button

背景: danger-600 / 文字: white

3. 線パターン

境界線・フロー線・支援線（点線）

Default Border

Strong Border

Support Edge (Dashed)

4. アプリ共通設定（JSON）

`apps/web/public/palette.json` を読み込んで切替可能

{
  "defaultTheme": "sea",
  "themes": {
    "sea": { "tokens": { "--color-blue-600": "#20b2aa", "...": "..." } },
    "sea-dark": { "tokens": { "--color-blue-600": "#198f8a", "...": "..." } }
  }
}
